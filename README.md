Brief

A local gym has asked you to build a piece of software to help them to manage memberships, and register members for classes.

Requirements

The app should allow the gym to create and edit Members
The app should allow the gym to create and edit Classes
The app should allow the gym to book members on specific classes
The app should show a list of all upcoming classes
The app should show all members that are booked in for a particular class


The project must be built using only:

HTML / CSS
Python
Flask
PostgreSQL and the psycopg


How to use Program 

First create database called gym_manager using createdb gym_mamager

Drop and create new tables psql -d gym_manager -f db/gym_manager.sql

populate databases with data by running console.py

start flask using flask run in the terminal 

copyy the http link given and paste into a browser
